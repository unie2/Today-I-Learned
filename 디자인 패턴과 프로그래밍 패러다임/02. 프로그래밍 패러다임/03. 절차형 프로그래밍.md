### ✔ 절차형 프로그래밍
  - 로직이 수행되어야 할 연속적인 계산 과정으로 이루어져 있다.
  - 일이 진행되는 방식으로 그저 코드를 구현하기만 하면 된다.

#### 1. 장점
  - 코드의 가독성이 좋다.
  - 실행 속도가 빠르다.

#### 2. 단점
  - 모듈화하기가 어렵다.
  - 유지 보수성이 떨어진다.

````javascript
// 최대값 찾기
const ret = [1, 2, 3, 4, 5, 11, 12]
let a = 0
for (let i=0; i<ret.length; i++) {
  a = Math.max(ret[i], a)
}

console.log(a) // 12
````
